# 🧠 Anesthesia Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting the appropriate **type and dosage of anesthesia** using patient health data.
The goal is to assist medical professionals in making safer, data-driven decisions during surgical procedures.

---

## 🎯 Problem Statement

Incorrect anesthesia dosage can lead to serious complications.
This project aims to build a **machine learning model** that predicts optimal anesthesia requirements based on patient-specific features.

---

## 📊 Dataset Features

The model uses the following patient attributes:

* Age
* Weight
* Blood Pressure
* Heart Rate
* Medical History

---

## 🔍 Data Analytics (EDA)

* Data cleaning and preprocessing
* Handling missing values and outliers
* Exploratory Data Analysis (EDA)
* Correlation analysis to identify key influencing factors

---

## 🤖 Machine Learning Approach

* Problem Type:

  * Classification → Anesthesia Type
  * Regression → Dosage Prediction

* Models Used:

  * Logistic Regression
  * Decision Tree
  * Random Forest

---

## 📈 Model Evaluation

* Accuracy (for classification)
* Mean Squared Error (MSE) (for regression)
* Model comparison and performance analysis
